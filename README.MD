# 📊 Customer Churn Analysis & Prediction Model

## 📌 Project Overview
This project focuses on analyzing customer churn behavior and predicting customers who are likely to leave a company. The project combines **Data Analysis, SQL, Machine Learning, and Power BI Visualization** to generate meaningful business insights and customer retention strategies.

---

# 🎯 Project Objectives
✅ Analyze customer churn patterns  
✅ Identify important factors affecting churn  
✅ Build a Machine Learning prediction model  
✅ Create an interactive Power BI dashboard  
✅ Generate business insights for decision-making
